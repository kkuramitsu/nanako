# PythonユーザのためのNanako入門

Nanakoは、教育用のプログラミング言語です。
Nanakoは、日本語ベースの文法、限られた言語仕様により、
プログラミングやアルゴリズムの基礎を短期間に体験的に学べるようになっています。

この文書ではPythonユーザのためのNanakoの習得するコツを書いています。

## 主な特徴

### できること
- 変数、代入
- インクリメント/デクリメントのみ（四則演算は自分で実装）
- 配列操作（多次元配列可）
- 文字列（Unicode配列として扱う）
- 関数、再帰関数
- トップレベルの評価 => 暗黙的なprint
- doctest（`>>>` で期待値を記述）

### ステートメント
- 1行、1ステートメント 行末の; なし
- ブロックは、カーリーブレイス { }
- ルール: もし〜ならば { } そうでなければ {} # if/else 相当
- ループ: N回、くり返す {} # くり返しの回数を見積もる
- くり返しを抜ける # break 相当
- 入力 x に対し { }　# lambda x 相当
- xが答え # return 相当

## 基本文法の比較

### 演算子

Nanakoは、インクリメント/デクリメントだけ演算子使えます。

```python
# Python
x = 10
x += 1
x -= 1
```

```nanako
# Nanako
x = 10
xを増やす
xを減らす
```

### 配列

```python
# Python
arr = [1, 2, 3]
arr[0] = 10
arr.append(4)
length = len(arr)
```

```nanako
# Nanako
arr = [1, 2, 3]
arr[0] = 10
arrの末尾に4を追加する
length = |arr|
```

### ルール

Nanakoでは、条件分岐はルールと呼ぶ。

```nanako
# Nanako
もし xが5ならば、{
    y = 1
}
そうでなければ、{
    もし xが10以上ならば、{
        y = 2
    }
    そうでなければ、{
        y = 3
    }
}
```

- else 節相当(そうでなければ)は省略可能
- 比較演算子は、もしに続けて以下の通りにかける。
    - `xが5ならば` → `x == 5`
    - `xが5以外ならば` → `x != 5`
    - `xが5以上ならば` → `x >= 5`
    - `xが5以下ならば` → `x <= 5`
    - `xが5より大きいならば` → `x > 5`
    - `xが5より小さいならば` → `x < 5`

### ループ

```nanako
10回、くり返す {
    もし 条件が1ならば、{
        くり返しを抜ける
    }
}
```

- 無限ループはかけない
- あらかじめ十分なくり返し回数を見積もり、必要に応じ抜ける

### 関数

```nanako
足し算 = 入力 x, y に対し、{     # def たし算(x, y):
    y回、くり返す {             #   for _ in range(y):
        xを増やす              #     x+=1
    }
    xが答え                    #   return x
}

足し算(3, 5)                   # print(たし算(3, 5))
```

変数スコープ:
  ローカル変数: 関数単位のスコープ
　グローバル変数は、参照できるが変更は反映されない。
